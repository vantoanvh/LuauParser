CST Nodes

Concrete Syntax Tree nodes that preserve exact source formatting. Each CST node corresponds to an AST node and captures precise token positions.

CST nodes are stored separately from AST nodes in the cstNodes map. Use result.cstNodes[astNode] to retrieve the CST data for any AST node. CST capture is enabled via captureCST = true in parse options.

CstNode CST Union Type

Union of all CST node types. The specific type depends on the corresponding AST node's kind.

CST Nodes

Concrete Syntax Tree nodes provide precise positioning for all tokens and structural elements not captured in the AST.

CST nodes are stored externally to the AST to keep it lightweight. You can retrieve the CST for any AST node using the cstNodes map in the Result object:

local cst = result.cstNodes[astNode]

CST Expressions

CstExprConstantNumber
CSTEXPR

Stores the original source text for a numeric literal.

Fields
valuestring
The raw source text of the number (e.g., "0xFF" or "1.2e3").
CstExprConstantString
CSTEXPR

Detailed source information for string literals, including quote styles and raw content.

Fields
kind"CstExprConstantString"
sourceStringstring?optional
The original source string (optional depending on configuration).
quoteStylenumber
CST-specific quote style: 0=Single, 1=Double, 2=Raw, 3=Interp.
blockDepthnumber
Nesting level of equals signs in long brackets (e.g., [==[]==] is 2).

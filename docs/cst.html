CST Nodes

Concrete Syntax Tree nodes that preserve exact source positioning for tokens, including delimiters and keywords.

Fields

Position of the opening (.
closeParenPositionPosition
Position of the closing ).

CstExprConstantNumber
CST

A numeric literal with its original source string.

Fields

Full range.

The exact text of the number literal.
CstExprConstantString
CST

A string literal with its original source string and quoting detail.

Fields
sourceStringstring?optional
The raw string text (including quotes or brackets).
quoteStylenumber
Quote style: 0=Single, 1=Double, 2=Raw, 3=Interpolated.
blockDepthnumber
Nesting depth for raw strings (e.g., 2 for [==[).

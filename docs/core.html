Core Types

Fundamental data structures for representing source locations and identifiers.

Position
HELPER

Represents a position in the source code (1-indexed line and column). Used to pinpoint exact locations for error reporting and CST data.

Fields
linenumber
The 1-indexed line number in the source file.
columnnumber
The 1-indexed column number in the source file.
Location
HELPER

Represents a range in the source code from a beginning position to an ending position. Every AST node has a location that spans the source text it was parsed from.

Fields
beginPosition
The starting position of the range (inclusive).
endPosition
The ending position of the range (exclusive).
AstName
ASTHELPER

Represents a simple name/identifier in the AST. Used for variable names, function names, type names, etc.

Fields
valuestring
The raw string value of the identifier.
AstLocal
ASTHELPER

Represents a local variable declaration in the AST. Tracks the variable's scope and shadowing information.

Fields
namestring
The name of the local variable.
locationLocation
Source location of the variable declaration.
shadowAstLocal?optional
Reference to a shadowed local variable with the same name in an outer scope.
functionDepthnumber
The nesting depth of the function containing this local variable.
loopDepthnumber
The nesting depth of loops containing this local variable.
annotationAstType?optional
Optional type annotation for the local variable.

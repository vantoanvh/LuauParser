Statements

AST nodes representing actions, logic, and declarations in Luau.

AstStat
ASTSTAT

Union type representing all possible statement nodes in the AST. Statements are constructs that perform actions but don't produce values.

Structure & Control

AstStatBlock
ASTSTAT

A block of statements, representing a scope boundary. Used for function bodies, control structure bodies, and the top-level chunk.

Fields
kind"StatBlock"
The statement kind identifier.
locationLocation
The range spanning all statements in the block.
body{AstStat}
The list of statements contained within this block.
hasEndboolean
True if the block is explicitly terminated by an end keyword.
hasSemicolonboolean?optional
True if followed by an optional semicolon.
AstStatIf
ASTSTAT

An if statement: if condition then ... elseif ... else ... end.

Fields
kind"StatIf"
The statement kind identifier.
locationLocation
The entire range from if to end.
conditionAstExpr
The expression evaluated to determine which branch to execute.
thenbodyAstStatBlock
The block of code executed if the condition is truthy.
elsebodyAstStat?optional
Optional branch executed if the condition is falsy (can be another AstStatIf for elseif).
thenLocationLocation?optional
Location of the then keyword.
elseLocationLocation?optional
Location of the else or elseif keyword.

Loops

AstStatWhile
ASTSTAT

A while loop: while condition do ... end.

Fields
kind"StatWhile"
locationLocation
conditionAstExpr
The loop condition expression.
bodyAstStatBlock
The loop body block.
hasDoboolean
True if the do keyword is present.
doLocationLocation
Location of the do keyword.
AstStatFor
ASTSTAT

A numeric for loop: for var = from, to, step do ... end.

Fields
kind"StatFor"
locationLocation
varAstLocal
The loop variable.
fromAstExpr
The starting value expression.
toAstExpr
The ending value expression.
stepAstExpr?optional
Optional step value expression (defaults to 1).
bodyAstStatBlock
The loop body block.

A local variable declaration: local a, b = expr1, expr2.

Fields
kind"StatLocal"
locationLocation
vars{AstLocal}
The list of local variables being declared.
values{AstExpr}
The list of initializer expressions.
equalsSignLocationLocation?optional
Location of the = sign (if values are provided).
AstStatAssign
ASTSTAT

An assignment statement: var1, var2 = expr1, expr2.

Fields
kind"StatAssign"
locationLocation
vars{AstExpr}
The list of l-value expressions being assigned to.
values{AstExpr}
The list of expressions being assigned.

Fields

locationLocation

The name of the type alias.
nameLocationLocation
Location of the type name.
typeAstType
The type being aliased.

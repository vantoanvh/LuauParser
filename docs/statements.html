Statements

Abstract Syntax Tree nodes representing constructs that perform actions but do not produce values.

AstStatBlock
STAT

A block of statements, representing a scope boundary. Used for function bodies, control structure bodies, and the top-level chunk.

Fields
kind"StatBlock"
The specific statement kind.
locationLocation
Full range of the block.
body{AstStat}
The list of statements in this block.
hasEndboolean
True if the block is terminated by the end keyword.
hasSemicolonboolean?optional
True if followed by an optional semicolon.
AstStatIf
STAT

An if statement: if condition then ... elseif ... else ... end.

Fields
kind"StatIf"
The specific statement kind.
locationLocation
Full range of the if statement.
conditionAstExpr
The condition expression.
thenbodyAstStatBlock
The block executed when the condition is truthy.
elsebodyAstStat?optional
Optional else/elseif branch (can be another AstStatIf or AstStatBlock).
thenLocationLocation?optional
Location of the then keyword.
AstStatWhile
STAT

A while loop: while condition do ... end.

Fields
kind"StatWhile"
The specific statement kind.
locationLocation
Full range.
conditionAstExpr
The loop condition expression.
bodyAstStatBlock
The loop body block.
doLocationLocation
Location of the do keyword.
AstStatRepeat
STAT

A repeat-until loop: repeat ... until condition.

Fields
kind"StatRepeat"
The specific statement kind.
conditionAstExpr
The loop termination condition.
bodyAstStatBlock
The loop body block.
AstStatBreak
STAT

A break statement: break. Exits the innermost loop immediately.

AstStatContinue
STAT

A continue statement: continue (Luau extension). Skips to the next iteration of the innermost loop.

AstStatReturn
STAT

A return statement: return expr1, expr2, .... Returns values from the current function.

Fields
list{AstExpr}
The list of expressions to return.
AstStatExpr
STAT

An expression statement: evaluates an expression for its side effects (e.g., function calls).

Fields
exprAstExpr
The expression to evaluate.
AstStatLocal
STAT

A local variable declaration: local a, b = expr1, expr2.

Fields
vars{AstLocal}
The list of local variables being declared.
values{AstExpr}
The list of initializer expressions.
equalsSignLocationLocation?optional
Location of the = sign.
AstStatFor
STAT

A numeric for loop: for var = from, to, step do ... end.

Fields
varAstLocal
The loop variable.
fromAstExpr
The starting value expression.
toAstExpr
The ending value expression.
stepAstExpr?optional
Optional step value expression (defaults to 1).
bodyAstStatBlock
The loop body block.
AstStatForIn
STAT

A generic for loop: for var1, var2 in expr1, expr2 do ... end.

Fields
vars{AstLocal}
The loop variables.
values{AstExpr}
The iterator expressions.
bodyAstStatBlock
The loop body block.
AstStatAssign
STAT

An assignment statement: var1, var2 = expr1, expr2.

Fields
vars{AstExpr}
The list of l-value expressions being assigned to.
values{AstExpr}
The list of expressions being assigned.
AstStatCompoundAssign
STAT

A compound assignment statement: var += expr, var -= expr, etc.

Fields
opnumber
The binary operator code (Add, Sub, Mul, etc.).
varAstExpr
The l-value expression being modified.
valueAstExpr
The expression to apply with the operator.
AstStatFunction
STAT

A function declaration statement: function a.b.c:name(...) ... end.

Fields
nameAstExpr
The l-value expression where the function is stored.
funcAstExprFunction
The function implementation expression.
AstStatLocalFunction
STAT

A local function declaration: local function name(...) ... end.

Fields
nameAstLocal
The local variable holding the function.
funcAstExprFunction
The function implementation expression.
AstStatTypeAlias
STAT

A type alias declaration: type Name<T> = Type or export type Name = Type.

Fields
namestring
The name of the type alias.
generics{AstGenericType}
Generic type parameters.
typeAstType
The type being aliased.
exportedboolean
True if this is an export type declaration.
AstStatTypeFunction
STAT

A type function declaration (Luau extension): type function Name(...) ... end.

Fields
namestring
Name of the type function.
bodyAstExprFunction
The compile-time function body.
AstStatDeclareGlobal
STAT

A global variable declaration (definition file syntax): declare name: Type.

Fields
namestring
The name of the global variable.
typeAstType
The declared type of the global.
AstStatDeclareFunction
STAT

A function declaration (definition file syntax): declare function name(...).

Fields
namestring
The function name.
paramsAstTypeList
The parameter types.
retTypesAstTypePack
The return type pack.
AstStatDeclareExternType
STAT

An extern type declaration (definition file syntax): declare extern type Name.

Fields
namestring
The name of the extern type.
superNamestring?optional
Optional parent type name (inheritance).
AstStatError
STAT

An error statement node, created when parsing fails.

Fields
expressions{AstExpr}
Any expressions that were successfully parsed.
statements{AstStat}
Any statements that were successfully parsed.

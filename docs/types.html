Type Annotations

AST nodes representing Luau's gradual type system constructs.

AstType
ASTTYPE

Union type representing all possible type annotation nodes in the AST.

Basic Types

ASTTYPE

Fields

locationLocation

The name of the type being referenced.
nameLocationLocation
Location of the type name.

Location of the prefix.

Type arguments passed to generic types.

True if type parameters are provided (even if empty).

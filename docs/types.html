AstTypeReference
ASTType

A named type reference: TypeName or Module.TypeName<Params>.

Example
local x: number               -- simple reference
local y: Array<string>        -- generic reference
local z: Module.Type          -- prefixed reference

True if type parameters are provided (even if empty).

Type arguments passed to generic types.

A table type: { prop: Type, [Key]: Value }.

Example

AstTypeTypeof
ASTType

A typeof type: typeof(expr). Infers the type from the given expression.

Example
local original = {x = 1, y = 2}
local copy: typeof(original) = {x = 3, y = 4}
Fields
kind"TypeTypeof"
locationLocation
exprAstExpr

Helpers & Result

Miscellaneous structures used throughout the parser and the final output format.

AST Helpers

AstAttr
HELPER

Attributes applied to functions or type definitions (e.g., @native, @checked).

Example
@native
local function fastLoop()
    -- optimized with native codegen
end
Fields
namestring
The attribute name (e.g., "native", "checked").
locationLocation
Source location of the attribute.
AstTableProp
HELPER

A property definition in a table type.

Fields
namestring
locationLocation
typeAstType
accessPositionPosition?optional
Position of the read or write keyword.
AstTableIndexer
HELPER

An indexer definition in a table type: [KeyType]: ValueType.

AstArgumentName
HELPER

A named argument in a function type declaration.

CST Helpers

CstQuotes
HELPER

Enum for string quoting styles: 0=Single, 1=Double, 2=Raw, 3=Interpolated.

Parser Metadata

Comment
HELPER

A comment found in the source text.

Fields
locationLocation
Range of the comment.
HotComment
HELPER

A "hot comment" (e.g., --!strict) that affects parsing or analysis.

Example
--!strict
--!native
local x: number = 10
Error
HELPER

A parse error entry.

Fields
locationLocation
messagestring

Configuration & Result

ParseOptions
HELPER

Options passed to the parser to control behavior.

Example
local result = Syntax.parse(code, {
    showComments = true,
    captureCST = true
})
Fields
showCommentsboolean?optional
True to include comments in the result.
captureCSTboolean?optional
True to build and return the CST.
Result
HELPER

The final output structure returned by the parser.

Example
local result = Syntax.parse("local x = 1")

print(result.root.kind)      --> "StatBlock"
print(#result.errors)        --> 0
print(#result.root.body)     --> 1
Fields
rootAstStatBlock
The top-level AST node (block of statements).
cstNodes{ [AstNode]: CstNode }?optional
Mapping from AST nodes to their corresponding CST data.
errors{Error}
List of all parse errors found.
comments{Comment}?optional
List of comments found in the source.
hotComments{HotComment}
List of hot comments found.

Helpers & Result

Supporting types and the final structure returned by the parser.

Result
HELPER

The object returned by the LuauParser.parse function. Contains the root AST block, captured CST data, and any errors encountered during parsing.

Fields
rootAstStatBlock
The root of the Abstract Syntax Tree.
errors{ParseError}
A list of all syntax errors found. The parser is resilient and will attempt to produce a root AST even if errors occur.
cstNodes{[any]: any}
A mapping from AST nodes to their corresponding CST data. Use this for precise source location tasks.
comments{Comment}
A list of all comments found in the source code.
ParseError
HELPER

Represents a syntax error encountered during parsing.

Fields
locationLocation
messagestring
A human-readable error message.
Comment
HELPER

Source information for comments found in the Luau code.

Fields

contentstring
The raw content of the comment.

Helpers & Result

Supporting structures, function attributes, and the final parser result type.

AST Support

AstAttr
HELPER

A function attribute: @native, @checked, @deprecated, etc.

Fields
typestring
The attribute kind (e.g., "Native", "Checked", "Deprecated").
args{AstExpr}
The list of arguments passed to the attribute.
namestring?optional
The raw attribute name as it appeared in the source.

Parser Result

Result
HELPER

The final result of parsing a Luau source file. Contains the AST root, comments, and metadata.

Fields
rootAstStatBlock
The root AST node, representing the top-level block of the script.
errors{ParseError}
A list of all parse errors encountered during execution.
hotcomments{HotComment}
Directives like --!strict or --!nonstrict.
cstNodeMapMap
A map linking AST nodes to their corresponding CST data nodes.
ParseError
HELPER

Represents a parse error with location and descriptive message.

Fields
locationLocation
The source location where the error occurred.
messagestring
The human-readable error message.

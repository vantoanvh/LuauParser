Expressions

AST nodes that evaluate to a value, including literals, variables, and function calls.

AstExpr
ASTEXPR

Union type representing all possible expression nodes in the AST. Expressions are constructs that evaluate to a value.

Literal Constants

AstExprConstantNil
ASTEXPR

The nil literal constant. Represents the absence of a value in Luau.

Fields
kind"ExprConstantNil"
The expression kind identifier.
locationLocation
The source location of the nil token.
AstExprConstantBool
ASTEXPR

A boolean literal constant: true or false.

Fields
kind"ExprConstantBool"
The expression kind identifier.
locationLocation
The source location of the boolean token.
valueboolean
The boolean value (true or false).
AstExprConstantNumber
ASTEXPR

A numeric literal constant (integer or floating-point). The original source text is stored in the CST node.

Fields
kind"ExprConstantNumber"
The expression kind identifier.
locationLocation
The source location of the number literal.
valuenumber
The parsed numeric value.
parseResultstring
Parsing status: "Ok" = success, "Imprecise" = exceeds 2^53, "HexOverflow" = hex > 64 bits, "BinOverflow" = binary > 64 bits, "Malformed" = parse failure.
AstExprConstantString
ASTEXPR

A string literal constant (single-quoted, double-quoted, or long bracket string).

Fields
kind"ExprConstantString"
The expression kind identifier.
locationLocation
The source location of the string literal.
valuestring
The processed string value (with escape sequences resolved).
quoteStylenumber
The style of quotes: 0=Double/Interp, 1=Single, 2=Raw Brackets, 3=Unquoted (table keys).

Variables & Scope

AstExprLocal
ASTEXPR

A reference to a local variable.

Fields
kind"ExprLocal"
The expression kind identifier.
locationLocation
The location of the variable name in this expression.
localAstLocal
The AstLocal object representing the referenced variable definition.
upvalueboolean
True if this is an upvalue (captured from an outer function scope).
AstExprGlobal
ASTEXPR

A reference to a global variable.

Fields
kind"ExprGlobal"
The expression kind identifier.
locationLocation
The source location of the global name.
namestring
The name of the global variable.

Indexing & Calls

AstExprCall
ASTEXPR

A function call expression: func(args) or obj:method(args).

Fields
kind"ExprCall"
The expression kind identifier.
locationLocation
The location spanning the entire call expression.
funcAstExpr
The expression being called (function or callable object).
args{AstExpr}
The list of argument expressions.
selfboolean
True if this is a method call using : syntax.
argLocationLocation
The location spanning the argument list (including parentheses).
AstExprIndexName
ASTEXPR

A named field access expression: expr.name or expr:name.

Fields
kind"ExprIndexName"
The expression kind identifier.
locationLocation
The entire indexed expression range.
exprAstExpr
The expression being indexed.
indexstring
The name of the field being accessed.
indexLocationLocation
The location of the field name.
opPositionPosition
The source position of the . or : operator.
opnumber
The operator character code (46 for ., 58 for :).

Operators

AstExprUnary
ASTEXPR

A unary operator expression: -expr, not expr, or #expr.

Fields
kind"ExprUnary"
The expression kind identifier.
locationLocation
The range spanning the operator and its operand.
opnumber
The unary operator code: 0=not, 1=minus, 2=length.
exprAstExpr
The operand expression.
AstExprBinary
ASTEXPR

A binary operator expression: left op right.

Fields
kind"ExprBinary"
The expression kind identifier.
locationLocation
The range spanning both operands and the operator.
opnumber
The binary operator code (Add, Sub, Mul, comparison, logical, etc.).
leftAstExpr
The left operand expression.
rightAstExpr
The right operand expression.

Advanced Expressions

AstExprTable
ASTEXPR

A table constructor expression: { items }.

Fields
kind"ExprTable"
The expression kind identifier.
locationLocation
The location spanning the entire table constructor.
items{AstExprTableItem}
The list of table items (key-value pairs or list elements).
AstExprFunction
ASTEXPR

A function expression (anonymous function or lambda): function(...) ... end.

Fields
kind"ExprFunction"
The expression kind identifier.
locationLocation
The entire range from function to end.
bodyAstStatBlock
The block of statements forming the function body.
args{AstLocal}
The list of function parameters.
varargboolean
True if the function accepts a variable number of arguments (...).
selfAstLocal?optional
The implicit self parameter (present for method definitions).

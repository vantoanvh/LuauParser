Expressions

Abstract Syntax Tree nodes representing constructs that evaluate to a value.

AstExprGroup
EXPR

A parenthesized expression: (expr). Used to override operator precedence or for clarity.

Fields
kind"ExprGroup"
The specific expression kind.
locationLocation
Source location of the parenthesized expression.
exprAstExpr
The inner expression wrapped in parentheses.
AstExprConstantNil
EXPR

The nil literal constant. Represents the absence of a value in Luau.

Fields
kind"ExprConstantNil"
The specific expression kind.
locationLocation
Source location of the nil literal.
AstExprConstantBool
EXPR

A boolean literal constant: true or false.

Fields
kind"ExprConstantBool"
The specific expression kind.
locationLocation
Source location of the boolean literal.
valueboolean
The boolean value (true or false).
AstExprConstantNumber
EXPR

A numeric literal constant (integer or floating-point).

Fields
kind"ExprConstantNumber"
The specific expression kind.
locationLocation
Source location of the numeric literal.
valuenumber
The parsed numeric value.
parseResultstring
Parsing status: "Ok", "Imprecise", "HexOverflow", "BinOverflow", or "Malformed".
AstExprConstantString
EXPR

A string literal constant (single-quoted, double-quoted, or long bracket string).

Fields
kind"ExprConstantString"
The specific expression kind.
locationLocation
Source location of the string literal.
valuestring
The processed string value (with escape sequences resolved).
quoteStylenumber
The quote style: 0=Simple, 1=Single, 2=Raw, 3=Unquoted identifier key.

AstExprCall
EXPR

A function call expression: func(args) or obj:method(args).

Fields
kind"ExprCall"
The specific expression kind.
locationLocation
Full location of the call expression.
funcAstExpr
The expression being called.
args{AstExpr}
The list of argument expressions.
selfboolean
True if this is a method call using : syntax.
argLocationLocation
The location spanning the argument list.
typeArguments{AstTypeOrPack}?optional
Optional explicit type arguments for generic functions.
AstExprIndexName
EXPR

A named field access expression: expr.name or expr:name.

Fields
kind"ExprIndexName"
The specific expression kind.
locationLocation
Full location of the index expression.
exprAstExpr
The expression being indexed.
indexstring
The name of the field being accessed.
indexLocationLocation
The location of the field name.
opPositionPosition
The position of the . or : operator.
opnumber
The operator character code (46 for ., 58 for :).

kind"ExprIndexExpr"
The specific expression kind.
locationLocation
Full location of the index expression.
exprAstExpr
The expression being indexed.
indexAstExpr
The expression used as the index/key.
AstExprFunction
EXPR

A function expression (anonymous function or lambda): function(...) ... end.

Fields
kind"ExprFunction"
The specific expression kind.
locationLocation
The full source range of the function.
attributes{AstAttr}
Function attributes like @native, @checked, @deprecated.
generics{AstGenericType}
Generic type parameters (e.g., <T, U>).
genericPacks{AstGenericTypePack}
Generic type pack parameters (e.g., <T...>).
selfAstLocal?optional
The implicit self parameter for methods (nil for regular functions).
args{AstLocal}
The list of function parameters.
returnAnnotationAstTypePack?optional
Optional return type annotation.
varargboolean
True if the function accepts varargs ....
varargLocationLocation
Location of the ... token.
varargAnnotationAstTypePack?optional
Optional type annotation for varargs.
bodyAstStatBlock
The function body (block of statements).
functionDepthnumber
Nesting depth of this function.
debugnamestring
Debug name for the function (inferred from context).
AstExprTable
EXPR

A table constructor expression: { items }.

Fields
kind"ExprTable"
The specific expression kind.
locationLocation
Full range of the table constructor.
items{AstExprTableItem}
The list of table items (key-value pairs or list elements).
AstExprTableItem
HELPER

Represents a single item in a table constructor.

Fields
kind"List" | "Record" | "General"
The item kind: "List" (array element), "Record" (named key), or "General" (bracketed key).
keyAstExpr?optional
The key expression (nil for List items).
valueAstExpr
The value expression.

The specific expression kind.

falseExprAstExpr

Expressions

Abstract Syntax Tree nodes representing constructs that evaluate to a value.

AstExpr AST Union Type

Union type representing all possible expression nodes in the AST. Expressions are constructs that evaluate to a value.

AstExprGroup
ASTExpression

A parenthesized expression: (expr). Used to override operator precedence or for clarity.

Example
local x = (a + b) * c
local y = (condition)
Fields
kind"ExprGroup"
locationLocation
Source location spanning the entire parenthesized expression.
exprAstExpr
The inner expression wrapped in parentheses.
AstExprConstantNil
ASTExpressionLiteral

The nil literal constant. Represents the absence of a value in Luau.

Example
local x = nil
if value == nil then end
Fields
kind"ExprConstantNil"
locationLocation
Source location of the nil keyword.
AstExprConstantBool
ASTExpressionLiteral

A boolean literal constant: true or false.

Example
local enabled = true
local disabled = false
Fields
kind"ExprConstantBool"
locationLocation
Source location of the boolean keyword.
valueboolean
The boolean value (true or false).
AstExprConstantNumber
ASTExpressionLiteral

A numeric literal constant (integer or floating-point). The original source text is stored in the CST node (CstExprConstantNumber.value).

Example
local a = 42
local b = 3.14159
local c = 0xFF
local d = 0b1010
local e = 1.5e10
Fields
kind"ExprConstantNumber"
locationLocation
Source location of the number literal.
valuenumber
The parsed numeric value.
parseResultstring
Parsing status: "Ok" = success, "Imprecise" = exceeds 2^53 (loses f64 precision), "HexOverflow" = hex literal exceeds 64 bits, "BinOverflow" = binary literal exceeds 64 bits, "Malformed" = failed to parse.
AstExprConstantString
ASTExpressionLiteral

A string literal constant (single-quoted, double-quoted, or long bracket string). The original source string (before escape processing) is stored in CstExprConstantString.sourceString.

Example
local a = "double quoted"
local b = 'single quoted'
local c = [[long bracket
multiline string]]
local d = [=[raw string with = signs]=]
Fields
kind"ExprConstantString"
locationLocation
Source location of the string literal.
valuestring
The processed string value (with escape sequences resolved).
quoteStylenumber
The quote style: 0=QuotedSimple (double quotes or interp string), 1=QuotedSingle (single quotes), 2=QuotedRaw (long brackets [[]]), 3=Unquoted (identifier-style key in table constructor like {foo = 1}).
AstExprLocal
ASTExpressionVariable

A reference to a local variable.

Example
local x = 10
print(x)  -- 'x' is AstExprLocal
Fields
kind"ExprLocal"
locationLocation
Source location of the variable reference.
localAstLocal
The AstLocal representing the referenced variable.
upvalueboolean
True if this is an upvalue (captured from an outer scope).
AstExprGlobal
ASTExpressionVariable

A reference to a global variable.

Example
print("hello")  -- 'print' is AstExprGlobal
game.Players   -- 'game' is AstExprGlobal
Fields
kind"ExprGlobal"
locationLocation
Source location of the global reference.
namestring
The name of the global variable.
AstExprVarargs
ASTExpression

A reference to the varargs ... in a vararg function. Only valid inside functions declared with ... parameter.

Example
local function sum(...)
    local args = {...}  -- '...' is AstExprVarargs
    return table.concat(args)
end
Fields
kind"ExprVarargs"
locationLocation
Source location of the ... token.
AstExprCall
ASTExpression

A function call expression: func(args) or obj:method(args).

Example
print("hello")          -- regular call
object:Method(a, b)     -- method call (self = true)
doSomething {x = 1}     -- table call
require "module"        -- string call
Fields
kind"ExprCall"
locationLocation
Source location spanning the entire call expression.
funcAstExpr
The expression being called (function or callable object).
args{AstExpr}
The list of argument expressions.
selfboolean
True if this is a method call using : syntax.
argLocationLocation
The location spanning the argument list (including parentheses).
typeArguments{AstTypeOrPack}?optional
Optional explicit type arguments for generic functions.
AstExprIndexName
ASTExpression

A named field access expression: expr.name or expr:name.

Example
local x = table.key      -- op = '.'
local y = object:Method  -- op = ':'
Fields
kind"ExprIndexName"
locationLocation
Source location spanning the entire index expression.
exprAstExpr
The expression being indexed.
indexstring
The name of the field being accessed.
indexLocationLocation
The location of the field name.
opPositionPosition
The position of the . or : operator.
opnumber
The operator character code (46 for ., 58 for :).
AstExprIndexExpr
ASTExpression

A bracketed index expression: expr[index].

Example
local x = array[1]
local y = dict["key"]
local z = matrix[row][col]
Fields
kind"ExprIndexExpr"
locationLocation
Source location spanning the entire index expression.
exprAstExpr
The expression being indexed.
indexAstExpr
The expression used as the index/key.
AstExprFunction
ASTExpression

A function expression (anonymous function or lambda): function(...) ... end.

Example
local add = function(a, b) return a + b end

local callback = function(value: T): T
    return value
end

table.sort(list, function(a, b)
    return a < b
end)
Fields
kind"ExprFunction"
locationLocation
Source location spanning the entire function expression.
attributes{AstAttr}
Function attributes like @native, @checked, @deprecated.
generics{AstGenericType}
Generic type parameters (e.g., <T, U>).
genericPacks{AstGenericTypePack}
Generic type pack parameters (e.g., <T...>).
selfAstLocal?optional
The implicit self parameter for methods (nil for regular functions).
args{AstLocal}
The list of function parameters.
returnAnnotationAstTypePack?optional
Optional return type annotation.
varargboolean
True if the function accepts varargs ....
varargLocationLocation
Location of the ... token (if vararg).
varargAnnotationAstTypePack?optional
Optional type annotation for varargs.
bodyAstStatBlock
The function body (block of statements).
functionDepthnumber
Nesting depth of this function.
debugnamestring
Debug name for the function (inferred from context).
argLocationLocation?optional
Location of the parameter list.
AstExprTable
ASTExpression

A table constructor expression: { items }.

Example
local array = {1, 2, 3}
local dict = {key = "value", another = 42}
local mixed = {1, 2, name = "mixed", ["computed"] = true}
Fields
kind"ExprTable"
locationLocation
Source location spanning the entire table constructor.
items{AstExprTableItem}
The list of table items (key-value pairs or list elements).
AstExprTableItem
ASTHelper

Represents a single item in a table constructor.

Fields
kind"List" | "Record" | "General"
"List" for array-style value, "Record" for key = value, "General" for [key] = value.
keyAstExpr?optional
The key expression (nil for List items).
valueAstExpr
The value expression.
AstExprUnary
ASTExpressionOperator

A unary operator expression: -expr, not expr, or #expr.

Example
local negative = -value      -- minus (op=1)
local inverted = not flag    -- not (op=0)
local length = #array        -- length (op=2)
Fields
kind"ExprUnary"
locationLocation
Source location spanning the entire unary expression.
opnumber
The unary operator (0=not, 1=minus, 2=length).
exprAstExpr
The operand expression.
AstExprBinary
ASTExpressionOperator

A binary operator expression: left op right. Covers arithmetic, comparison, logical, and concatenation operators.

Example
local sum = a + b        -- arithmetic
local cmp = x > y        -- comparison
local and_ = p and q     -- logical
local str = s1 .. s2     -- concatenation
Fields
kind"ExprBinary"
locationLocation
Source location spanning the entire binary expression.
opnumber
The binary operator code (see BinaryOp enum in parser).
leftAstExpr
The left operand expression.
rightAstExpr
The right operand expression.
AstExprTypeAssertion
ASTExpressionType

A type assertion expression: expr :: Type. Used to assert that an expression has a specific type.

Example
local n = value :: number
local s = data :: string
local any = obj :: any
Fields
kind"ExprTypeAssertion"
locationLocation
Source location spanning the entire assertion.
exprAstExpr
The expression being asserted.
annotationAstType
The type being asserted.
AstExprInstantiate
ASTExpressionType

An explicit type instantiation expression: expr<T, U>. Used to explicitly specify type arguments for a generic function or type.

Example
local result = cast(value)
local array = Array.new()
Fields
kind"ExprInstantiate"
locationLocation
Source location spanning the entire instantiation.
exprAstExpr
The expression being instantiated.
typeArguments{AstTypeOrPack}
The type arguments provided.
AstExprIfElse
ASTExpressionControl Flow

An if-else expression: if condition then trueExpr else falseExpr. The expression form of conditional logic (not the statement form).

Example
local max = if a > b then a else b
local result = if x then "yes" elseif y then "maybe" else "no"
Fields
kind"ExprIfElse"
locationLocation
Source location spanning the entire if-else expression.
conditionAstExpr
The condition expression.
hasThenboolean
True if the then keyword is present.
trueExprAstExpr
Expression evaluated when condition is truthy.
hasElseboolean
True if the else keyword is present.
falseExprAstExpr
Expression evaluated when condition is falsy.
AstExprInterpString
ASTExpressionLiteral

An interpolated string expression: `text {expr} more text`. Allows embedding expressions directly within string literals.

Example
local name = "World"
local greeting = `Hello, {name}!`
local math = `Result: {a + b}`
Fields
kind"ExprInterpString"
locationLocation
Source location spanning the entire interpolated string.
strings{string}
The literal string segments between expressions.
expressions{AstExpr}
The embedded expressions.
AstExprError
ASTExpressionError

An error expression node, created when parsing fails. Used to allow partial parsing to continue after errors.

Fields
kind"ExprError"
locationLocation
Source location of the error.
expressions{AstExpr}
Any expressions that were successfully parsed.
messageIndexnumber
Index into the error messages table.

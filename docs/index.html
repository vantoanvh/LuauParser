Luau Parser API

Comprehensive documentation for the Luau Abstract Syntax Tree (AST) and Concrete Syntax Tree (CST) nodes.

This documentation details the internal structures used by the Luau parser to represent and process Luau source code. It is designed for developers building tools like linters, formatters, and static analyzers.

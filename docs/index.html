Luau Parser API

A high-performance, lossless parser for the Luau programming language. This documentation covers the AST and CST structures returned by the parser.

The parser produces a rich Abstract Syntax Tree (AST) for semantic analysis and an optional Concrete Syntax Tree (CST) for source-accurate transforms. All nodes include detailed location mapping.

Getting Started

The parser's main entry point returns a Result object containing the root AST block and any errors encountered. You can traverse the AST to perform analysis or use the CST mapping for code formatting tools.

-- Example Usage
local Syntax = require("Syntax")
local result = Syntax.parse("local x: number = 10")

if #result.errors == 0 then
    print("Parsed " .. result.root.kind)
end

Luau Parser API

A high-performance, lossless parser for the Luau programming language. This documentation covers the AST and CST structures returned by the parser.

The parser produces a rich Abstract Syntax Tree (AST) for semantic analysis and an optional Concrete Syntax Tree (CST) for source-accurate transforms. All nodes include detailed location mapping.

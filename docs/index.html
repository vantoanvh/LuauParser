Luau Parser API

Complete documentation for the Luau Parser AST and CST types.

Overview

This documentation covers the Abstract Syntax Tree (AST), Concrete Syntax Tree (CST), and helper structures used in the Luau Parser.

Pages
  • Core Types - Base structures like Position, Location, and AST Unions.
  • Expressions - AST nodes for expressions (AstExpr).
  • Statements - AST nodes for statements (AstStat).
  • Type Annotations - AST nodes for type annotations (AstType).
  • CST Nodes - Concrete Syntax Tree nodes for full fidelity parsing.
  • Helpers & Result - Helper structures and parser result types.

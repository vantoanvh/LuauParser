üöÄ Quick Start

Here is the most basic example of how to convert a string into an AST.

-- 1. Require the module
local Parser = require(path.to.Parser)

local source = "print('Hello World')"

-- 2. Create instance
local parserInstance = Parser.new(source)

-- 3. Parse
local ast = parserInstance:Parse()
print(ast)
